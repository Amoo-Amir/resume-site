مستندات API - سیستم مدیریت فروشگاه آنلاین

این API یک فروشگاه آنلاین کامل را مدیریت می‌کند. تمام اندپوینت‌ها نیاز به توکن JWT دارند.

احراز هویت

POST /api/v1/auth/login 
{
    "email": "user@example.com",
    "password": "yourpassword"
}

پاسخ موفق:

{
    "token": "eyJhbGciOiJIUzI1NiIs...",
    "user": {
        "id": 1,
        "name": "کاربر نمونه",
        "email": "user@example.com"
    }
}

مدیریت محصولات

GET /api/v1/products

دریافت لیست تمام محصولات با قابلیت صفحه‌بندی

پارامترهای اختیاری: page, limit, category, sort

GET /api/v1/products/{id}

دریافت جزئیات یک محصول خاص

POST /api/v1/products

ایجاد محصول جدید (فقط ادمین)

{
    "name": "نام محصول",
    "price": 250000,
    "category_id": 1,
    "stock": 100,
    "description": "توضیحات محصول"
}

سبد خرید

GET /api/v1/cart

دریافت محتویات سبد خرید کاربر جاری

POST /api/v1/cart/add 
{
    "product_id": 5,
    "quantity": 2
}

پرداخت

POST /api/v1/checkout 
{
    "cart_id": 123,
    "payment_method": "online",
    "address_id": 5
}

پاسخ موفق:

{
    "order_id": 456,
    "status": "pending",
    "payment_url": "https://payment.gateway.com/pay/..."
}
