درباره من

من امیرمهدی رحمتی هستم، توسعه‌دهنده بک‌اند در سطح جونیور با اشتیاق فراوان برای یادگیری و ساختن سیستم‌های کارآمد.

تخصص من در طراحی و پیاده‌سازی RESTful API، آشنایی با معماری میکروسرویس، بهینه‌سازی دیتابیس و اصول امنیت API است. همچنین با Docker برای کانتینری کردن برنامه‌ها کار می‌کنم و همیشه به دنبال بهترین practices در توسعه نرم‌افزار هستم.

تاکنون دو پروژه عملی کامل طراحی و پیاده‌سازی کرده‌ام: سیستم پشتیبانی (Support Ticket System) و سیستم رزرو نوبت (Appointment Booking System). این پروژه‌ها شامل مستندات کامل API، احراز هویت امن و معماری تمیز هستند. من به دنبال اولین تجربه کاری حرفه‌ای در تیمی پویا هستم تا بتوانم ارزش آفرینی کنم و همزمان از همتیمی‌های با تجربه یاد بگیرم.

سیستم مدیریت تیکت پشتیبانی

Node.js Express.js MongoDB Mongoose JWT

API کامل برای ثبت و مدیریت تیکت‌های پشتیبانی. کاربران می‌توانند تیکت ثبت کنند و ادمین‌ها قادر به پاسخ‌دهی و تغییر وضعیت تیکت‌ها هستند.

  • احراز هویت و نقش‌های کاربر/ادمین با JWT
  • رمزنگاری امن رمزعبور با bcrypt
  • مدیریت متغیرهای محیطی با dotenv
  • راه‌اندازی خودکار با nodemon در محیط توسعه

سیستم رزرو نوبت

Node.js Express.js MongoDB Mongoose JWT

API رزرو نوبت که کاربران می‌توانند بر اساس سرویس مورد نظر، زمان‌های خالی را مشاهده و نوبت خود را ثبت کنند.

  • احراز هویت امن با JWT و رمزنگاری bcrypt
  • نمایش زمان‌های خالی بر اساس سرویس انتخاب شده
  • جلوگیری از دابل بوکینگ (عدم تداخل زمانی نوبت‌ها)
  • لغو و ویرایش نوبت‌های ثبت شده
  • دریافت تاریخچه رزروهای هر کاربر
  • اعتبارسنجی داده‌ها با کتابخانه Joi
